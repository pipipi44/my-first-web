**MindAnchor** is a serene, meaningful digital space designed to help you track your moods, reflect on your thoughts, and anchor your mind in the present moment. Built with modern web technologies, it offers a beautiful, glassmorphism-inspired interface for your personal digital gardening.

## âœ¨ Features

- **Mood Tracking**: Record your daily moods with a timestamped log.
- **Mood Analytics**: Visual dashboard showing your mood balance and activity statistics.
- **Privacy First**: All data is stored locally in your browser (`localStorage`).
- **Dynamic Greeting**: Time-aware greetings (Morning, Afternoon, Evening, Night).
- **Interactive UI**:
  - Glassmorphism design system.
  - Smooth animations and transitions.
  - "Privacy Blur" mode for your entries.
  - Typing effect intro.
- **Themes**: Multiple color themes (Default, Forest, Ocean).
- **Internationalization (i18n)**: Support for English and Chinese.

## ğŸš€ Getting Started

### Prerequisites

You don't need any complex backend. A modern web browser is all you need.

### Installation / Running Locally

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/YOUR_USERNAME/mind-anchor.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd mind-anchor
    ```
3.  **Open `index.html`** directly in your browser, or use a local server:
    ```bash
    # If you have Node.js installed
    npx serve .
    ```
